# Branding

Brand assets for Immich ReverseGeo.

## Files

- `logo-mark.svg` — standalone symbol, transparent background (use anywhere)
- `logo-horizontal.svg` — symbol + wordmark on dark background (banners, headers)
- `logo-stacked.svg` — symbol + stacked wordmark on dark background (badges, splash)

## Design

**Concept**: a classic map pin whose circular head is divided into 5 pie segments — one per official Immich brand color. The center carries a white ring, dark field, and blue location dot to reference both a camera lens and a map marker.

**Symbol structure**:
- 5 pie segments (clockwise from top, 72° each): Red → Orange → Green → Blue → Pink
- Pin tail: deep navy `#1E3A8A`
- Center: `#FFFFFF` → `#1E293B` → `#1E83F7`

## Official Immich brand colors used

## Supporting colors

- Background dark: `#0F172A`
- Pin tail / center dark: `#1E3A8A` / `#1E293B`
- Text primary: `#E2E8F0`
